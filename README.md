# Academic Class Database System

A web-based class scheduling and database management system built as part of an Information Systems course project.

## 🚀 Features
- Store and manage class schedules
- Organize subject data
- Manage student/class information
- Simple interface for academic tracking

## 🛠 Tech Stack
- JavaScript
- HTML
- CSS

## 🎯 Purpose
This project was developed to simulate a simple academic database system for managing class schedules and related data.
